# Fake-Glitch-Virus
THIS DOES NOT DO ANYTHING, JUST A FUN TRICK TO PULL ON YOUR FRIENDS

(Mac) COMMANDS YOU NEED TO USE:
# Install Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Git
brew install git

# Install Python
brew install python

# Download the project
git clone https://github.com/ZyntrioxYT/Fake-Glitch-Virus.git

# Open the project folder
cd YOUR_REPOSITORY

# Upgrade pip
python3 -m pip install --upgrade pip

# Install required packages
python3 -m pip install pygame

# Install PyInstaller
python3 -m pip install pyinstaller

# Build the app
pyinstaller --onefile --windowed --name Roblox glitch_demo.py glitch_demo.py

# Run the app
python3 glitch_demo.py
